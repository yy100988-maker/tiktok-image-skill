# tiktok-image-skill

Vetech AI 的 TikTok 电商图像生成 Skill 仓库。

## 内容

- `output/image-skill/`：可直接分发给其他 agent 的标准 Skill 包
- `output/image-skill.zip`：打包压缩文件，便于下载和分发

## 功能

- 图像理解
- 单张生图
- 目录批量生图
- Excel / CSV 批量生图
- 多国家语言适配
- TikTok Shop 目标场景示例

## 使用前提

- 需要配置 `IMG_API_KEY` 或 `API_KEY`
- 需要本地 `python`
- 参考图默认使用 base64 传输

## 目录

- `output/image-skill/SKILL.md`
- `output/image-skill/scripts/`
- `output/image-skill/references/`

## 下载

可直接使用仓库中的 `output/image-skill/` 目录，或下载 `output/image-skill.zip`。
